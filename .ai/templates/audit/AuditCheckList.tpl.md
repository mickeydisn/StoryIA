## 1. COHÉRENCE NARRATIVE

### 1.1 Cohérence Informationnelle
- [ ] Tous les concepts introduits sont-ils liés aux prérequis établis?
- [ ] Les nouveaux concepts sont-ils compatibles avec l'univers établi?
- [ ] Y a-t-il des contradictions avec des scènes précédentes?
- [ ] Les connaissances des personnages sont-elles cohérentes avec leur parcours?
- [ ] Les informations données sont-elles au bon niveau de complexité pour ce moment?
- [ ] Les retours en arrière (flashbacks) sont-ils cohérents avec les faits établis?
- [ ] Les informations révélées étaient-elles déjà implicitement présentes?

### 1.2 Cohérence Causale
- [ ] La chaîne cause → effet est-elle claire et logique?
- [ ] Chaque action a-t-elle une conséquence observable?
- [ ] Les délais (temporels, physiques) sont-ils respectés?
- [ ] Les décisions des personnages découlent-elles logiquement de leurs connaissances?
- [ ] Si on supprime cette scène, y a-t-il rupture dans la causalité globale?
- [ ] Les événements simultanés sont-ils compatibles entre eux?
- [ ] Les actions ont-elles des conséquences non anticipées mais logiques?

### 1.3 Cohérence d'État
- [ ] L'état d'entrée correspond-il à l'état de sortie de la scène précédente?
- [ ] Toutes les transformations d'état sont-elles justifiées?
- [ ] Les ressources (carburant, énergie, temps, argent) sont-elles trackées correctement?
- [ ] L'état émotionnel évolue-t-il de manière crédible?
- [ ] Les relations entre personnages suivent-elles une progression logique?
- [ ] L'état physique des personnages (fatigue, blessures) est-il cohérent?
- [ ] Les changements d'environnement sont-ils justifiés et trackés?

### 1.4 Cohérence Spatiotemporelle
- [ ] La localisation est-elle claire à tout moment?
- [ ] Les déplacements respectent-ils les contraintes physiques (vitesse, distance)?
- [ ] Le temps écoulé est-il cohérent avec les actions décrites?
- [ ] Y a-t-il synchronisation avec les événements parallèles?
- [ ] Les délais de communication (latence, vitesse lumière) sont-ils respectés?
- [ ] Les fuseaux horaires et décalages temporels sont-ils gérés correctement?
- [ ] La durée des actions correspond-elle à leur description?
- [ ] Les cycles jour/nuit, saisons sont-ils cohérents?

### 1.5 Cohérence Conceptuelle (Registre SF)
- [ ] Les concepts scientifiques introduits sont-ils compatibles entre eux?
- [ ] Les extrapolations sont-elles cohérentes avec la science établie?
- [ ] Les inventions respectent-elles les règles de l'univers?
- [ ] Les limitations techniques sont-elles constantes?
- [ ] Y a-t-il des deus ex machina scientifiques non justifiés?
- [ ] Les termes techniques sont-ils utilisés de manière cohérente?
- [ ] Les niveaux technologiques sont-ils homogènes (ou les écarts justifiés)?

## 2. RIGUEUR SCIENTIFIQUE (Hard SF - Asimov)

### 2.1 Exactitude Scientifique
- [ ] Les lois physiques fondamentales sont-elles respectées?
- [ ] Les ordres de grandeur sont-ils corrects?
- [ ] Les calculs mentionnés sont-ils vérifiables?
- [ ] Les termes techniques sont-ils utilisés correctement?
- [ ] Les extrapolations sont-elles plausibles et expliquées?
- [ ] Les unités de mesure sont-elles cohérentes et appropriées?
- [ ] Les phénomènes décrits respectent-ils les principes établis?
- [ ] Les impossibilités physiques sont-elles évitées (sauf justification)?

### 2.2 Problème & Solution
- [ ] Le problème technique est-il clairement posé?
- [ ] Les contraintes sont-elles explicites?
- [ ] La solution découle-t-elle logiquement des prémisses?
- [ ] Y a-t-il un coût/compromis à la solution?
- [ ] Les conséquences imprévues sont-elles crédibles?
- [ ] Les alternatives ont-elles été envisagées (implicitement ou explicitement)?
- [ ] La solution utilise-t-elle les ressources disponibles de manière réaliste?
- [ ] L'échec est-il une option crédible?

### 2.3 Intégration Narrative
- [ ] La science est-elle montrée via l'action plutôt qu'expliquée?
- [ ] Les explications techniques sont-elles nécessaires au récit?
- [ ] Évite-t-on l'infodump et le technobabble gratuit?
- [ ] Les personnages interagissent-ils naturellement avec la technologie?
- [ ] La technologie crée-t-elle du conflit/tension?
- [ ] Les concepts complexes sont-ils rendus accessibles sans simplification abusive?
- [ ] La rigueur scientifique renforce-t-elle l'immersion?

### 2.4 Crédibilité de l'Univers
- [ ] Les détails techniques renforcent-ils l'immersion?
- [ ] L'environnement obéit-il à des règles constantes?
- [ ] Les limitations créent-elles des choix difficiles?
- [ ] Y a-t-il des détails "inutiles" qui font vrai?
- [ ] L'univers pourrait-il fonctionner sans l'histoire?
- [ ] Les implications sociétales de la technologie sont-elles visibles?
- [ ] L'infrastructure technique est-elle cohérente (maintenance, logistique)?
- [ ] Les facteurs humains (erreur, fatigue) sont-ils pris en compte?

## 3. STRUCTURE & TEMPO (Gravity Falls)

### 3.1 Hook & Ouverture
- [ ] La première phrase accroche-t-elle immédiatement?
- [ ] Le lecteur sait-il où on est et ce qui se passe en 2 paragraphes?
- [ ] Y a-t-il une question/tension dès le début?
- [ ] Évite-t-on les démarrages mous ou descriptifs?
- [ ] L'ouverture établit-elle le ton de la scène?
- [ ] Y a-t-il un contraste ou une surprise dans l'ouverture?

### 3.2 Rythme & Progression
- [ ] Le rythme change-t-il au moins une fois dans la scène?
- [ ] Y a-t-il alternance entre tension et respiration?
- [ ] Chaque paragraphe fait-il avancer l'action/révélation/émotion?
- [ ] Les moments lents sont-ils justifiés (construction de tension)?
- [ ] Y a-t-il un crescendo vers le climax de scène?
- [ ] Les longueurs de paragraphes varient-elles pour créer du rythme?
- [ ] Le rythme correspond-il à l'état émotionnel des personnages?
- [ ] Les ellipses temporelles sont-elles efficaces?

### 3.3 Surprises & Retournements
- [ ] Y a-t-il au moins un élément inattendu?
- [ ] Les révélations sont-elles préparées mais surprenantes?
- [ ] Évite-t-on la prévisibilité?
- [ ] Les twists sont-ils fair-play (indices plantés)?
- [ ] Y a-t-il un moment "oh merde!" ou "oh wow!"?
- [ ] Les subversions d'attentes sont-elles satisfaisantes?
- [ ] Les fausses pistes sont-elles crédibles?

### 3.4 Clôture & Transition
- [ ] La scène se termine-t-elle sur une accroche?
- [ ] Le lecteur veut-il savoir ce qui suit?
- [ ] La transition vers la scène suivante est-elle fluide?
- [ ] Laisse-t-on des questions ouvertes stratégiques?
- [ ] Évite-t-on les fins plates ou résolutives?
- [ ] La dernière phrase a-t-elle de l'impact?
- [ ] Le cliffhanger (si présent) est-il justifié et efficace?

### 3.5 Gestion du Mystère
- [ ] Des indices sont-ils plantés pour plus tard?
- [ ] Y a-t-il des détails qui prendront sens en relecture?
- [ ] Le mystère s'approfondit-il sans frustrer?
- [ ] Les révélations partielles alimentent-elles la curiosité?
- [ ] Y a-t-il équilibre entre réponses et nouvelles questions?
- [ ] Les red herrings sont-ils subtils et crédibles?
- [ ] Le mystère sert-il l'intrigue ou est-il artificiel?

### 3.6 Architecture de Scène
- [ ] La scène a-t-elle une structure claire (setup/confrontation/résolution)?
- [ ] Le climax de scène est-il clairement identifiable?
- [ ] Les micro-tensions internes à la scène sont-elles efficaces?
- [ ] La scène a-t-elle sa propre complétude tout en s'intégrant au tout?
- [ ] Les beats narratifs sont-ils bien placés?

## 4. STYLE & VOIX (Damasio)

### 4.1 Langue & Singularité
- [ ] La langue est-elle travaillée, non-conventionnelle?
- [ ] Y a-t-il une voix distinctive reconnaissable?
- [ ] Les phrases ont-elles du rythme et de la musicalité?
- [ ] Évite-t-on les clichés et formules toutes faites?
- [ ] Le vocabulaire est-il riche et précis?
- [ ] Les néologismes (si présents) sont-ils justifiés et compréhensibles?
- [ ] La syntaxe exprime-t-elle une personnalité narrative?

### 4.2 Sensorialité & Immersion
- [ ] Au moins 3 sens sont-ils sollicités?
- [ ] Les sensations sont-elles spécifiques (pas génériques)?
- [ ] L'environnement SF a-t-il une texture unique?
- [ ] Peut-on "sentir" l'espace, la gravité, l'air, la température?
- [ ] Les détails sensoriels servent-ils l'émotion?
- [ ] Les sensations physiques des personnages sont-elles présentes?
- [ ] L'environnement est-il perçu par le corps, pas seulement l'œil?

### 4.3 Expérimentation Formelle
- [ ] Y a-t-il des prises de risque stylistiques?
- [ ] La forme épouse-t-elle le fond (chaos → phrases chaotiques)?
- [ ] Les jeux typographiques sont-ils justifiés et lisibles?
- [ ] Les ruptures de forme servent-elles le récit?
- [ ] Évite-t-on la gratuité expérimentale?
- [ ] Les changements de perspective narrative sont-ils maîtrisés?
- [ ] Les expérimentations renforcent-elles l'immersion ou la brisent-elles?

### 4.4 Poésie & Beauté
- [ ] Y a-t-il au moins une image forte/mémorable?
- [ ] La science génère-t-elle du sublime?
- [ ] Les métaphores sont-elles originales et justes?
- [ ] Y a-t-il des phrases qui donnent envie d'être relues?
- [ ] La beauté coexiste-t-elle avec la rigueur technique?
- [ ] Les descriptions transcendent-elles le purement fonctionnel?
- [ ] Y a-t-il des moments de grâce stylistique?

### 4.5 Flux & Conscience
- [ ] Les pensées des personnages sont-elles organiques?
- [ ] Les dialogues sonnent-ils naturels (pas artificiels)?
- [ ] Le flux narratif reflète-t-il l'état émotionnel?
- [ ] Y a-t-il des moments d'intériorité profonde?
- [ ] Évite-t-on l'exposition déguisée en pensée?
- [ ] Le monologue intérieur est-il distinct de la narration?
- [ ] Les ruptures de pensée sont-elles crédibles?

## 5. PERSONNAGES & PSYCHOLOGIE

### 5.1 Arc de Personnage
- [ ] Chaque personnage présent évolue-t-il (même minime)?
- [ ] Les transformations sont-elles crédibles?
- [ ] Les décisions révèlent-elles quelque chose de nouveau?
- [ ] Y a-t-il tension entre désir et devoir/contrainte?
- [ ] Les vulnérabilités sont-elles montrées sans faiblesse gratuite?
- [ ] Les changements sont-ils progressifs ou justement brusques?
- [ ] Les personnages apprennent-ils de leurs erreurs?

### 5.2 Cohérence Psychologique
- [ ] Les réactions émotionnelles sont-elles justes?
- [ ] Les personnages agissent-ils selon leur caractère établi?
- [ ] Les changements de cap sont-ils motivés?
- [ ] Évite-t-on les comportements "pour les besoins du scénario"?
- [ ] Les limites cognitives/émotionnelles sont-elles respectées?
- [ ] Les traumas et blessures passées influencent-ils le comportement?
- [ ] Les personnages ont-ils des angles morts psychologiques cohérents?
- [ ] Les mécanismes de défense sont-ils crédibles?

### 5.3 Dynamique Relationnelle
- [ ] Les relations évoluent-elles via cette scène?
- [ ] Les dialogues révèlent-ils les tensions sous-jacentes?
- [ ] Y a-t-il subtexte dans les échanges?
- [ ] Les conflits sont-ils authentiques (pas forcés)?
- [ ] Les alliances/ruptures sont-elles méritées?
- [ ] Les dynamiques de pouvoir sont-elles claires?
- [ ] Les non-dits sont-ils aussi importants que les dits?
- [ ] Les relations ont-elles une histoire qui se ressent?

### 5.4 Compétence & Crédibilité
- [ ] Les expertises des personnages sont-elles cohérentes?
- [ ] Font-ils des erreurs humaines même en étant experts?
- [ ] Les non-experts posent-ils des questions pertinentes?
- [ ] Évite-t-on les personnages trop parfaits ou trop idiots?
- [ ] Les compétences servent-elles l'histoire (pas juste cool)?
- [ ] Les limites de compétence créent-elles du conflit?
- [ ] L'apprentissage est-il progressif et crédible?

### 5.5 Voix & Perspective
- [ ] Chaque personnage a-t-il une voix intérieure distinctive?
- [ ] Le point de vue choisi est-il le plus efficace pour cette scène?
- [ ] Les biais cognitifs du personnage POV influencent-ils la narration?
- [ ] Les filtres culturels/personnels sont-ils présents?
- [ ] Le lecteur perçoit-il des choses que le personnage POV ignore?

## 6. THÉMATIQUE & PROFONDEUR

### 6.1 Exploration Thématique
- [ ] Le thème principal est-il présent (même subtilement)?
- [ ] Y a-t-il une question philosophique/éthique posée?
- [ ] Le dilemme SF fait-il écho à notre réalité?
- [ ] Évite-t-on le message trop appuyé/moralisateur?
- [ ] La science questionne-t-elle l'humanité?
- [ ] Les thèmes sont-ils explorés par l'action, pas le discours?
- [ ] Y a-t-il des nuances, des zones grises thématiques?

### 6.2 Symbolisme & Résonance
- [ ] Y a-t-il des symboles visuels/conceptuels?
- [ ] Les métaphores fonctionnent-elles à plusieurs niveaux?
- [ ] Les détails ont-ils une charge thématique?
- [ ] La scène résonne-t-elle au-delà de l'immédiat?
- [ ] Y a-t-il des échos avec d'autres œuvres (hommages)?
- [ ] Les motifs récurrents sont-ils présents et enrichis?
- [ ] Le symbolisme est-il subtil ou trop appuyé?

### 6.3 Subversion & Originalité
- [ ] Les attentes du genre sont-elles challengées?
- [ ] Évite-t-on les tropes SF éculés?
- [ ] Y a-t-il un angle neuf sur un concept classique?
- [ ] Les personnages échappent-ils aux stéréotypes?
- [ ] La scène surprend-elle même les lecteurs SF aguerris?
- [ ] Les subversions sont-elles motivées ou gratuites?
- [ ] L'originalité sert-elle le propos ou est-elle une fin en soi?

### 6.4 Complexité & Nuance
- [ ] Les problèmes moraux ont-ils plusieurs facettes?
- [ ] Évite-t-on le manichéisme simpliste?
- [ ] Les conséquences sont-elles ambiguës plutôt que claires?
- [ ] Les personnages ont-ils des positions défendables même antagonistes?
- [ ] La complexité enrichit-elle sans confondre?

## 7. WORLDBUILDING & IMMERSION

### 7.1 Show Don't Tell
- [ ] L'univers est-il révélé via l'action/interaction?
- [ ] Les détails techniques émergent-ils naturellement?
- [ ] Évite-t-on les paragraphes d'exposition pure?
- [ ] Les personnages ne s'expliquent pas l'évident entre eux?
- [ ] L'environnement "parle" par lui-même?
- [ ] Les infos sont-elles distillées progressivement?
- [ ] Le lecteur déduit-il plutôt qu'on ne lui explique?

### 7.2 Cohérence Universelle
- [ ] Les règles de cet univers sont-elles respectées?
- [ ] Les technologies ont-elles des conséquences sociales visibles?
- [ ] L'économie/politique sous-jacente est-elle logique?
- [ ] Les détails contradictoires sont-ils éliminés?
- [ ] L'univers "fonctionne" indépendamment du protagoniste?
- [ ] Les institutions et organisations sont-elles crédibles?
- [ ] Les conflits d'intérêts et de ressources sont-ils logiques?

### 7.3 Détails Évocateurs
- [ ] Y a-t-il des petits détails qui font "vrai"?
- [ ] L'usure, la patine, l'histoire sont-elles présentes?
- [ ] Les objets/lieux ont-ils une identité?
- [ ] Évite-t-on le décor générique "vaisseau spatial standard"?
- [ ] Y a-t-il des détails "inutiles" qui enrichissent?
- [ ] Les objets ont-ils une histoire visible?
- [ ] Les imperfections et dysfonctionnements sont-ils présents?

### 7.4 Culture & Société
- [ ] Les différences culturelles sont-elles visibles et cohérentes?
- [ ] Le langage reflète-t-il les structures sociales?
- [ ] Les tabous et normes sociales sont-ils présents?
- [ ] L'histoire de cet univers est-elle palpable?
- [ ] Les différences générationnelles sont-elles marquées?

### 7.5 Échelle & Perspective
- [ ] Les ordres de grandeur (spatial, temporel) sont-ils bien rendus?
- [ ] Le vertige de l'immensité (espace, temps) est-il présent?
- [ ] La place de l'humain dans l'univers est-elle questionnée?
- [ ] Les dimensions cosmiques sont-elles tangibles?

## 8. TECHNIQUE D'ÉCRITURE

### 8.1 Clarté & Lisibilité
- [ ] Sait-on toujours qui fait quoi?
- [ ] Les antécédents des pronoms sont-ils clairs?
- [ ] Les transitions temporelles/spatiales sont-elles évidentes?
- [ ] Évite-t-on les phrases ambiguës?
- [ ] Un lecteur distrait peut-il suivre?
- [ ] Les scènes d'action sont-elles chorégraphiées clairement?
- [ ] Les descriptions complexes sont-elles compréhensibles?

### 8.2 Économie & Précision
- [ ] Chaque mot est-il nécessaire?
- [ ] Les adverbes sont-ils évités au profit de verbes forts?
- [ ] Les descriptions sont-elles concises mais évocatrices?
- [ ] Y a-t-il du gras à couper?
- [ ] Les répétitions sont-elles intentionnelles (effet) ou parasites?
- [ ] Le mot juste est-il choisi plutôt que plusieurs approximations?
- [ ] Les périphrases sont-elles évitées?

### 8.3 Variété & Rythme
- [ ] Les longueurs de phrases varient-elles?
- [ ] Les structures syntaxiques sont-elles diversifiées?
- [ ] Y a-t-il alternance entre action et introspection?
- [ ] Les dialogues et narration s'équilibrent-ils?
- [ ] Le rythme sert-il l'émotion du moment?
- [ ] Les phrases courtes créent-elles de l'urgence quand nécessaire?
- [ ] Les phrases longues permettent-elles l'immersion contemplative?

### 8.4 Dialogues
- [ ] Chaque personnage a-t-il une voix distinctive?
- [ ] Les dialogues font-ils avancer l'intrigue?
- [ ] Y a-t-il du sous-texte (on ne dit pas tout)?
- [ ] Évite-t-on les dialogues "as you know, Bob"?
- [ ] Les échanges sonnent-ils naturels à voix haute?
- [ ] Les silences et hésitations sont-ils significatifs?
- [ ] Les dialogues révèlent-ils le caractère autant que l'info?
- [ ] Les interruptions et chevauchements sont-ils naturels?

### 8.5 Point de Vue & Focalisation
- [ ] Le point de vue est-il maintenu de manière cohérente?
- [ ] Les changements de POV sont-ils clairs et justifiés?
- [ ] La distance narrative (proche/éloignée) est-elle appropriée?
- [ ] Évite-t-on le head-hopping non maîtrisé?
- [ ] La focalisation révèle-t-elle la psyché du personnage?

### 8.6 Temporalité Narrative
- [ ] Les temps verbaux sont-ils cohérents?
- [ ] Les flashbacks sont-ils clairement signalés?
- [ ] Les ellipses temporelles sont-elles efficaces?
- [ ] Le rythme narratif correspond-il au temps diégétique?
- [ ] Les anticipations (flash-forward) sont-elles maîtrisées?

## 9. INTÉGRATION GLOBALE

### 9.1 Liens Structurels
- [ ] Cette scène est-elle reliée à au moins 2 autres?
- [ ] Y a-t-il des motifs récurrents (callbacks)?
- [ ] Les parallèles/échos sont-ils intentionnels?
- [ ] La scène s'inscrit-elle dans un arc plus large?
- [ ] Pourrait-on la retirer sans dommage? (Si oui → problème)
- [ ] Les setups antérieurs trouvent-ils des payoffs?
- [ ] De nouveaux setups sont-ils établis pour plus tard?

### 9.2 Progression Narrative
- [ ] Le statut quo est-il modifié par cette scène?
- [ ] L'intrigue avance-t-elle concrètement?
- [ ] De nouvelles possibilités narratives s'ouvrent-elles?
- [ ] Le lecteur en sait-il plus qu'avant (info/émotion)?
- [ ] Y a-t-il momentum vers le climax global?
- [ ] Les enjeux sont-ils rehaussés ou complexifiés?
- [ ] Des portes se ferment-elles (réduction des options)?

### 9.3 Équilibre Global
- [ ] Cette scène maintient-elle l'équilibre action/réflexion du livre?
- [ ] Le ratio technique/humain est-il approprié?
- [ ] La tonalité est-elle cohérente avec l'ensemble?
- [ ] Le POV utilisé est-il le bon pour cette scène?
- [ ] La longueur est-elle proportionnée à l'importance?
- [ ] La scène respecte-t-elle le "contrat" passé avec le lecteur?
- [ ] L'équilibre entre les arcs narratifs est-il maintenu?

### 9.4 Foreshadowing & Payoff
- [ ] Des éléments sont-ils préparés pour des révélations futures?
- [ ] Les Chekhov's guns antérieurs sont-ils utilisés?
- [ ] Les indices sont-ils suffisamment subtils?
- [ ] Les payoffs sont-ils satisfaisants et surprenants?
- [ ] La relecture enrichira-t-elle la compréhension de cette scène?

### 9.5 Architecture Narrative
- [ ] La scène occupe-t-elle la bonne position dans la structure?
- [ ] Contribue-t-elle à l'acte dans lequel elle se trouve?
- [ ] Les points de plot majeurs sont-ils respectés?
- [ ] La tension globale progresse-t-elle de manière appropriée?
- [ ] La scène prépare-t-elle efficacement les suivantes?

---

## 10. IMPACT & MÉMORABILITÉ

### 10.1 Moment Signature
- [ ] Y a-t-il au moins un moment mémorable dans cette scène?
- [ ] Une image/réplique/révélation restera-t-elle gravée?
- [ ] Le lecteur pourra-t-il résumer cette scène en une phrase forte?
- [ ] Y a-t-il un "moment de cinéma" (visuel fort)?

### 10.2 Résonance Émotionnelle
- [ ] La scène provoque-t-elle une émotion claire?
- [ ] L'impact émotionnel est-il mérité (pas manipulateur)?
- [ ] Y a-t-il des moments de catharsis ou de tension?
- [ ] Le lecteur est-il émotionnellement investi?

### 10.3 Satisfaction Intellectuelle
- [ ] La scène offre-t-elle matière à réflexion?
- [ ] Les révélations sont-elles élégantes?
- [ ] Le lecteur se sent-il intelligent en déduisant des choses?
- [ ] La complexité est-elle gratifiante plutôt que frustrante?

### 10.4 Valeur de Relecture
- [ ] La scène gagne-t-elle en profondeur à la relecture?
- [ ] Des détails prennent-ils un nouveau sens?
- [ ] L'ironie dramatique fonctionne-t-elle?
- [ ] Y a-t-il des layers d'interprétation?

## 11. BASE DE DONNÉES & WORLDBUILDING (Database Audit)

### 11.1 Cohérence des Entrées
- [ ] Chaque entrée suit-elle le format standard établi?
- [ ] Les informations sont-elles organisées de manière logique?
- [ ] Y a-t-il des incohérences entre entrées liées?
- [ ] Les descriptions sont-elles complètes sans être verbeuses?
- [ ] Les références croisées sont-elles bidirectionnelles?
- [ ] Les entrées respectent-elles la hiérarchie conceptuelle?

### 11.2 Références Croisées & Navigation
- [ ] Tous les liens [[XXX]] pointent-ils vers des entrées existantes?
- [ ] Les références sont-elles pertinentes et non redondantes?
- [ ] Y a-t-il des liens manquants entre éléments interconnectés?
- [ ] Les références utilisent-elles la syntaxe standardisée?
- [ ] Les liens créent-ils un réseau navigable cohérent?
- [ ] Évite-t-on les références circulaires inutiles?

### 11.3 Intégration Chronologique
- [ ] Les références temporelles pointent-elles vers des périodes définies?
- [ ] Les événements sont-ils correctement situés dans la timeline?
- [ ] Les évolutions historiques sont-elles cohérentes?
- [ ] Les références temporelles utilisent-elles la nomenclature standard?
- [ ] Y a-t-il des conflits temporels entre entrées?
