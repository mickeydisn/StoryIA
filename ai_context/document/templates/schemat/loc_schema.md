# Location Database Schema

This schema defines the required sections and structure for location database entries (LOC_*.md files).

## Required Sections

### Basic Information
- **Location Name**
- **Type**
- **Scale**

### Physical Description
- **Environment**
- **Sensory**

### Plot Integration
- **First Appearance**
- **Key Scenes**

### Thematic Significance
- **Symbolic Meaning**

### Quality Assurance
- **Consistency Check**
