# Character Database Schema

This schema defines the required sections and structure for character database entries (CHAR_*.md files).

## Required Sections

### Basic Information
- **Full Name**
- **Aliases/Titles**
- **Pronunciation**
- **Age**
- **Gender**
- **Appearance**
- **Health Status**

### Archetype & Role
- **Primary Archetype**
- **Secondary Archetypes**
- **Role in Story**
- **POV Status**

### Background & Origin
- **Birthplace**
- **Family**
- **Education/Training**
- **Occupation/Profession**
- **Social Status**

### Personality & Psychology
- **Core Personality Traits**
- **Strengths**
- **Flaws/Weaknesses**
- **Motivations**
- **Fears**
- **Values/Beliefs**

### Expertise & Skills
- **Technical Expertise**
- **Knowledge Domains**
- **Skills**
- **Inexperience/Ignorance**

### Character Arc
- **Starting State**
- **Key Milestones**
- **Transformation Points**
- **Ending State**
- **Arc Satisfaction**

### Relationships
- **Primary Relationships**
- **Secondary Relationships**
- **Antagonistic Relationships**

### Story Integration
- **First Appearance**
- **Chapter Appearances**
- **POV Chapters**
- **Key Scenes**

### Thematic Role
- **Themes Embodied**
- **Symbolic Meaning**
- **Narrative Function**

### Development Tracking
- **Consistency Checkpoints**
- **Evolution Markers**
- **Relationship Milestones**

### Cultural & Diversity Notes
- **Cultural Background**
- **Representation**
- **Sensitivity Considerations**

### Plot Integration Notes
- **Plot Threads Connected**
- **Conflicts Driven**
- **Resolutions Involved**

### Voice & Communication
- **Speech Patterns**
- **Communication Style**
- **Language/Accent**

### Visual/Iconic Elements
- **Signature Items**
- **Iconic Moments**
- **Visual Motifs**

### Database Cross-References
- **Related Concepts**
- **Related Locations**
- **Related Characters**
- **Related Systems**
- **Timeline References**

### Quality Assurance
- **Functionality Check**
- **Necessity Check**
- **Balance Check**
- **Integration Check**
