## Basic Information
- **Concept Name**: Data Compression Fundamentals
- **Category**: Computer Science / Information Theory
- **Alternative Names**: Data Encoding, Information Compression, Lossless/Lossy Compression

## Definition & Description
- **Core Definition**: Techniques for reducing the size of data while preserving essential information, enabling efficient storage and transmission.
- **Extended Description**: Data compression algorithms identify and eliminate redundancy in data, representing information using fewer bits. Lossless compression preserves all original data, while lossy compression sacrifices some information for greater compression ratios. Modern compression uses entropy coding, dictionary methods, and transform coding.
- **Key Characteristics**:
  - Redundancy elimination
  - Entropy-based encoding
  - Lossless vs. lossy trade-offs
  - Compression ratio optimization
- **Scope**: Fundamental data representation and storage techniques
