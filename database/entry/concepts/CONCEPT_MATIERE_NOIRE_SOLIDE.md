# Concept Database Entry - [[CONCEPT_MATIERE_NOIRE_SOLIDE]]

## Basic Information
- **Concept Name**: Matière Noire Solide (Solid Dark Matter)
- **Category**: Physics / Xenoarchaeology
- **Alternative Names**: Anomaly, Black Monolith, Exotic Matter

## Definition & Description
- **Core Definition**: An alien object discovered beneath the BNF that absorbs all light and violates known physics, appearing to be a message or artifact left by an unknown civilization.
- **Extended Description**: A tetrahedral black monolith floating in a preserved pre-Delestage lab. It absorbs all light (Vantablack natural), is near 0K, and causes non-Euclidean nausea. When touched, it transmits a vision: a burning galaxy and the message "Nous aussi" (We too). Memoria Dei cannot identify it—this breaks her omniscience.
