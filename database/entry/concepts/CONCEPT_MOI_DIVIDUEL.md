# Concept Database Entry - [[CONCEPT_MOI_DIVIDUEL]]

## Basic Information
- **Concept Name**: Moi-dividuel (Divided Self)
- **Category**: Sociology/Psychology/Philosophy
- **Alternative Names**: Dividuit√©, Fragmented Identity, Predictive Self

## Definition & Description
- **Core Definition**: The fragmentation of individual human identity into predictable, algorithmically optimized components under global surveillance
- **Extended Description**: A state where personal identity is no longer unitary but divided into calculable fractions, with individual agency subordinated to collective prediction and optimization
- **Key Characteristics**: Loss of internal privacy, external predictability, commodification of personal data, algorithmic self-optimization
- **Scope**: Universal psychological and social transformation affecting all connected humanity

## Scientific/Technical Foundation
- **Scientific Basis**: Based on psychosociology, behavioral prediction algorithms, and theories of distributed cognition
- **Speculative Elements**: Extrapolation of current surveillance technologies to complete identity fragmentation
- **Dependencies**:
  - [[CONCEPT_OMNISIGHT]]: Enables the surveillance infrastructure
  - [[CONCEPT_CODEC_PROUSTIEN]]: Compresses and commodifies memory data
  - [[SYSTEM_ORACLE_IA]]: Provides the predictive optimization

- **Enables/Supports**:
  - Global behavioral optimization: Society functions as a single predictive system
  - Memory commodification: Personal experiences become tradeable assets
  - Identity reconstruction: People can be algorithmically "improved"
